Mobile-first. J.A.R.V.I.S. for Android is a pure voice assistant built with Flutter/Dart. Say “Hey Jarvis” (or just “Jarvis”) to start continuous conversation mode and end it with “bye” or “goodbye”.
Inspired by Iron Man — Built for Real Life.
JARVIS can open the browser, tell weather and time, and handle general conversation. Memory lasts for the current session only — no database yet.
Terminal edition. A minimal desktop version runs directly in the terminal (requirements needed) and is voice-operated. Wake with “Jarvis”, chat hands-free, and use quick commands for utility tasks.
